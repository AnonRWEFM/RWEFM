# Riemannian Entropic Wasserstein Flow Matching

Flow Matching for distributions on General Geometries with the Riemannian Entropic Map

This implementation is written in Python3 and relies on FLAX, JAX, & JAX-OTT.

# Installation

To install JAX, simply run the command:

    pip install --upgrade "jax[cuda12_pip]" 
    
And to install RiemannianWassersteinFlowMatching along with the rest of the requirements, please clone this repo. For detailed guides, please follow provided notebooks in the examples folder.
